## 🚀 快速开始

### 数据处理系统
1. 访问数据处理系统链接
2. 上传需要处理的数据文件
3. 选择相应的处理选项
4. 下载处理后的结果

### 部门分类系统
1. 进入部门分类系统
2. 配置分类规则（可选）
3. 上传待分类数据
4. 查看分类结果和统计报告

### 问题分析器 ✨**新功能使用指南**
1. **上传数据**
   - 支持Excel (.xlsx, .xls) 和CSV格式
   - 拖拽文件或点击选择
   - 确认问题列名称（默认检测"问题"列）

2. **配置分类**（可选但推荐）
   - 点击"🔥 加载海底捞预设"获取13种预定义类别
   - 或手动添加自定义类别和关键词
   - 设置停用词过滤无关词汇

3. **开始分析**
   - 点击"🚀 开始分析"按钮
   - 系统自动执行TF-IDF关键词提取
   - 智能问题分类和聚类分析

4. **查看结果**
   - 📊 统计概览：总问题数、类别数等
   - 🥇 热点排行榜：按问题数量排序的类别
   - 📈 可视化图表：分类分布和关键词重要度
   - 📝 详细分析：每个类别的关键词和代表问题

5. **导出报告**
   - 📥 一键下载Excel完整分析报告
   - 包含：分类统计、详细结果、关键词分析等多个工作表

## 🛠️ 技术架构

### 前端技术栈
- **框架**：原生JavaScript + HTML5 + CSS3
- **图表库**：Chart.js（分类分布、关键词分析）
- **文件处理**：SheetJS（Excel）、PapaParse（CSV）
- **UI设计**：响应式布局、渐变背景、现代化组件

### 核心算法
- **TF-IDF算法**：词频-逆文档频率计算，准确提取关键词
- **智能分词**：中文字符识别、滑动窗口分词
- **聚类算法**：基于关键词覆盖率的智能分类
- **权重优化**：长关键词权重加成、多重匹配评分

### 部署方式
- **托管平台**：GitHub Pages
- **CDN加速**：静态资源CDN分发
- **兼容性**：支持现代浏览器（Chrome、Firefox、Safari、Edge）
